RWD: Responsive Web Design

Objetivo: adaptar nuestros contenidos y presentación al dispositivo que utilice en cada momento el usuario.

Se puede centrar en el tamaño, pero también podría en el ancho de banda, etc.

Dos posibilidades:

  • Mejora progresiva: de movil a escritorio
  • Degradación progresiva: de escritorio a móvil

Para usar RWD necesitamos

  • usar CSS Media Query
  • introducir en la cabecera la etiqueta meta viewport para que los contenidos funcionen correctamente en móviles, tablets, etc..
    <meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">

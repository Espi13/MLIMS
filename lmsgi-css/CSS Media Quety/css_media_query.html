CSS3 Media Queries

CSS2 Media Types

La regla @media de CSS2 permite aplicar diferentes estilos para diferentes medios, pincipalmente screen y print como ya hemos visto con el uso de link

 
  @media print {
    body { font-size: 10pt;}
    aside { display: none;}
  }

O con @import

 
  @import url("estilos_impresora.css") print;

O con link 

 
  <link rel="stylesheet" type="text/css" media="print, handheld" href="especial.css" />

CSS3 Media Queries

Las Media Queries en CSS3 extienden la idea de las CSS2 media types: en vez de mirar sólo el tipo de dispositivo pueden consultar las características del dispositivo, por ejemplo la anchura y altura del dispositivo, la orientación, la anchura mínima o máxima, etc.

Sintaxis de una Media Query

Añade a la sintasis de una CSS2 media type una o más expresiones, las cuales dan como resultado un valor verdadero o falso. El resultado global es verdadero si el tipo coincide y la expresiones son verdaderas. Si es verdadera, las reglas de estilo incluidas se aplican.

El tipo del medio es opcional, si no se incluye vale "all". Por lo tanto estas reglas son equivalentes:

 
  @media (orientation: portrait) { … }
  @media all and (orientation: portrait) { … }

Ejemplos

  • Modificamos el color de fondo al alcanzar una anchura mínima Ejemplo 1
  • Si el ancho es suficiente el menú flota a la izquierdaEjemplo 2

O usando la orientación del dispositivo:

 
  body {
    background-color: lightgreen;
  }
  @media only screen and (orientation: landscape) {
    body {
      background-color: lightblue;
    }
  }

Nota: active el inspector de elementos, seleccione "Toggle device mode" (el icono del móvil) y seleccioné un dispositivo móvil. Modifique la orientación (portrait o landscape) y observe como se modifica el color de fondo. También si hace el ancho mayor que el alto..
